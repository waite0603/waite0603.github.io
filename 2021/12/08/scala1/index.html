Scala 环境搭建及部署

发表于2021-12-08|更新于2021-12-08
|字数总计:209|阅读时长:1分钟|阅读量:

Scala

  • Scala 基于 JVM, 与 Java 完全兼容, 比 Java 更加面对对象
  • Scala 是一门函数式编程语言
  • Scala 对集合类型数据处理有非常好的支持
  • Spark 底层为 Scala 编写

环境搭建

Win

  1. Jdk 1.8 安装成功
  2. 下载对应版本 Scala
  3. 解压, 配置环境变量, 在 Path 配置到 bin 目录

Idea 搭建 Scala 开发环境

  1. Setting -> Plugins -> 下载 Scala 插件

  2. New -> Project -> Maven -> 勾选 Create -> Next

  3. 项目右键 -> add Frameworks support -> 添加 Scala

  4. main -> 创建directory叫做scala -> 右键scala选择mark as source root

  5. scala -> New -> Scala -> Object
