MongoDB安装包下载

https://www.mongodb.com/try/download/community

Version: 第一位为主版本号, 第二位如果为 偶数 则为稳定版, 奇数为开发版, 第三位为修订号

Package: msi 为安装包, zip解压既可使用

启动 MongoDB

使用命令行启动

  1. 新建一个 data 文件夹用于存放 数据库文件 以及 日志文件, 并在 data 文件夹下创建 db 和 log 子文件夹, 其中 db: 存储数据库文件, log: 储存日志文件

  2. 在 mongo/bin 打开 cmd 执行

  3. 查看 data\log\mongodb.log, 显示 mongo 启动成功, 若想关闭则则直接关闭 cmd 窗口即可
