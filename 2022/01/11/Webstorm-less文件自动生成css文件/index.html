Webstorm--less文件自动生成css文件

发表于2022-01-11|更新于2022-01-11
|字数总计:103|阅读时长:1分钟|阅读量:

此方法需安装 nodejs 环境，在官网进行下载 —- nodejs下载

通过 npm 工具包安装** less，命令行 npm install less -g**；通过命令 lessc 测试 less 是否安装成功

在 webstorm 中打开 “File” → “Settings” → “Tools” → “File Watchers” ，弹出配置窗口。点击右上角 “+”，选择 Less ，如下图：

Program 绑定 lessc.cmd 所在位置

Arguments 绑定转化路径

重启 WebStorm 即可
